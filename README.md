# X32-M32-scn-to-xlsx
 
Reads a ".scn" file from a Midas M32/Berhinger X32 digital mixer, and outputs a Chanel Plot as a ".xlsx" file.
All input routing settings are suported.
The "Outputs" sections displays the "Out 1-16" routing settings.

To use X32-M32-scn-to-xlsx, download and run "[dist/X32-M32-scn-to-xlsx v0.7.exe](https://github.com/Ultraegern/X32-M32-scn-to-xlsx/blob/main/dist/X32-M32-scn-to-xlsx%20v0.7.exe)".
